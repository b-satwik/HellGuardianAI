# Project Description - HellGuardian AI OS

HellGuardian AI is an autonomous, Jarvis-inspired productivity operating system designed to shield users from missed deadlines, burnout, and scheduling conflicts. Built for the Google AI Hackathon under the **"Last-Minute Life Saver"** problem statement, it operates as an active supervisor.

---

## 🎯 The Vision: "Jarvis for students & professionals"

Unlike traditional productivity systems (passive todo apps, checklist managers, and notification hubs) that users easily swipe away and ignore, HellGuardian AI acts as a command center. It continually monitors:
*   **Google Calendar** agendas to flag conflicts and find focus windows.
*   **Gmail** threads to detect incoming task requests and deadlines.
*   **Google Tasks** lists to synchronize actionable priorities.
*   **Google Drive** files to summarize document outlines and details.

It coordinates this data through a custom **Gemini-powered multi-agent engine** to calculate burnout risks, construct optimized daily agendas, and dynamically guide the operator through focus blocks.

---

## 🎙️ Hands-Free Conversation

HellGuardian AI features a dedicated, browser-native **Voice Assistant** with Speech-to-Text and Text-to-Speech synthesis. 
Users can speak commands like *"Summarize today's emails"* or *"Create task from my latest gmail"* to trigger complex agentic workflows. A smart text-cleaning parser strips terminal decorations from syntheses, ensuring the spoken output is completely natural.

---

## 💻 Cyber SOC Aesthetic

The visual design is inspired by high-security Cyber Security Operations Center (SOC) dashboards. It features high-contrast cyber-cyan and green typography, live cognition logging streams, draggable command terminals, and equalizer waveforms. 
All design layouts are accessibility-audited, featuring steel blue text blocks on deep-space dark backgrounds to guarantee readability for evaluators.
